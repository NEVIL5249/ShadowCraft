``` bash
npm install @nevil5249/shadowcraft
```

------------------------------------------------------------------------

# Usage

## Tailwind v4 (Recommended)

Add to CSS:

``` css
@import "tailwindcss";
@plugin "@nevil5249/shadowcraft";
```

------------------------------------------------------------------------

## Tailwind v3

Add to tailwind.config.js:

``` js
module.exports = {
  plugins: [
    require('@nevil5249/shadowcraft')
  ],
}
```
